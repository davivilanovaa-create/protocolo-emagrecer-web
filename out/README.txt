=== PROTOCOLO EMAGRECER - INSTRUÇÕES HOSTINGER ===

PASSO 1: PREPARAR OS ARQUIVOS
- Todos os arquivos estão prontos nesta pasta
- Não é necessário fazer nada adicional

PASSO 2: FAZER UPLOAD NA HOSTINGER
1. Acesse o painel cPanel da Hostinger
2. Abra o File Manager
3. Navegue até "public_html" (raiz do seu domínio)
4. Faça upload de TODOS os arquivos desta pasta:
   - index.html
   - .htaccess (IMPORTANTE - arquivo oculto)
   - pasta assets/
   - pasta images/
   - Todas as imagens JPG

PASSO 3: VERIFICAR CONFIGURAÇÕES
1. Certifique-se de que .htaccess foi enviado
2. Verifique se mod_rewrite está ativado (padrão na Hostinger)
3. Acesse seu domínio

PASSO 4: TESTAR
- Acesse https://seudominio.com
- Teste navegações e rotas
- Teste Quiz Onboarding
- Teste Dashboard

IMPORTANTE:
⚠️ Site ESTÁTICO (frontend apenas)
⚠️ Backend (autenticação, banco de dados) NÃO funciona
⚠️ Para usar com backend, use Manus Hosting ou Railway/Render

DÚVIDAS?
Se rotas não funcionarem, verifique se .htaccess foi enviado.
